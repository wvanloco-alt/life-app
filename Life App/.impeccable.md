# Design Context — Life App

## Target Audience
Individual professionals and active people who want to organize their life holistically — managing goals, training plans, schedules, budgets, and personal development in one place. Used primarily on desktop, occasionally on tablet.

## Use Cases
- Planning weekly/monthly schedules around work, training, and personal goals
- Tracking progress on multi-sport training plans (climbing, tennis)
- Managing household budgets and spending
- Setting and reviewing life goals across different roles (athlete, professional, parent, etc.)

## Brand Personality / Tone
- **Calm and focused** — not flashy or gamified. The app should feel like a trusted personal planner, not a productivity hustle tool.
- **Warm and grounded** — approachable, not clinical. Earthy warmth over cold tech aesthetics.
- **Quietly confident** — professional quality without corporate stiffness.
- **Minimal but not sparse** — clean layouts with purposeful use of color and space. Information-dense where needed, breathing room where appropriate.

## Visual Direction
- Light mode primary, warm-tinted neutrals (oklch hues ~50 warm amber)
- Dark mode with matching warm neutrals
- Accent colors tied to user-defined roles (each role has its own color)
- shadcn/ui component library with Tailwind CSS v4
- CSS variables using oklch color format
- Typography: Plus Jakarta Sans (body/sans), Fraunces (display headings), JetBrains Mono (code/mono)
- Border radius: `0.625rem`
- Motion: `fade-up` / `fade-in` entrance animations with `prefers-reduced-motion` fallbacks
- Clean data presentation — tables, calendars, activity cards
- Lucide icons for navigation, no emojis in chrome
- No decoration for decoration's sake

## Technical Constraints
- Next.js 16 App Router with Turbopack
- Tailwind CSS v4 + shadcn/ui components
- next-themes for light/dark mode
- @dnd-kit/core for drag-and-drop (pointerWithin collision detection)
- Must work well on 1280px+ screens, functional on tablet
- Performance matters — SQLite backend, standalone Docker deployment
